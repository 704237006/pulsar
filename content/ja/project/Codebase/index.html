Pulsarのコードベース

Pulsarへの貢献をお望みの場合は、こちらにリポジトリ構成の概要があります

以下のパネルではPulsarのコードベースの中のいくつかのコアディレクトリについて説明します。

pulsar-broker

Pulsar Brokerに関連するコードです。

言語: Java

pulsar-common

Pulsarのメッセージプロトコルに関連するアセットです。

言語: Java, Protocol Buffers

dashboard

Pulsarダッシュボードを起動するWebクライアントです。

言語: Python

kubernetes

PulsarのためのKubernetesの設定です。

言語: YAML

docker

PulsarのDockerイメージを生成するためのアセットです。

言語: Dockerfile, shell, Python

managed-ledger

言語: Java

pulsar-broker-auth-athenz

pulsar-client

pulsar-spark

pulsar-storm

pulsar-testclient

src

言語: XML

protobuf

Protocol Bufferの定義からコードを生成するためのヘルプです。

all

Mavenのassemblyプラグインのための設定です。

言語: XML

bin

Pulsarの管理のための実行可能なスクリプトです。

言語: Bash

site

このWebサイトを生成するためのアセットです。

言語: Markdown, Sass, HTML, JavaScript

.travis

Pulsarが利用している継続的インテグレーションシステムのTravis CIの設定です。
