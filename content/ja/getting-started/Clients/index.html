Pulsarクライアントライブラリ

Pulsarのクライアントライブラリは現在3つの言語に対応しています:

  • Java
  • Python
  • C++

Javaクライアント

PulsarのJavaクライアントを使用してメッセージをproduce, consumeするためのチュートリアルとして、Pulsar Javaクライアントをご確認ください。

2つの独立した Javadoc API ドキュメントセットが利用可能です:

ライブラリ 利用目的
org.apache.pulsar.client.api Pulsarのトピック上でメッセージをproduce, consumeするために利用するPulsar Javaクライアント
org.apache.pulsar.client.admin Pulsar admin APIのJavaクライアント

Pythonクライアント

PulsarのPythonクライアントを使うためのチュートリアルとして、Pulsar Pythonクライアントをご確認ください。

pdocによって生成されたPythonクライアントのAPIドキュメントもあります。

C++クライアント

PulsarのC++クライアントを使うためのチュートリアルとして、Pulsar C++クライアントをご確認ください。

Doxygenによって生成されたC++クライアントのAPIドキュメントもあります。
